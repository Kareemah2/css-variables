# Playing with CSS Variables

A fun, minimalist project that demonstrates how to manipulate CSS custom properties dynamically using vanilla JavaScript.

## Features
* Live Updates: Changes to spacing, blur, and base colors reflect instantly on the page.
* Pure JS: Uses `document.documentElement.style.setProperty` to update variables without a framework.
* Modern CSS: Powered entirely by native CSS variables.

## Usage
Open `index.html` in any browser and use the control knobs to adjust the image styling.
